# 🌾 Flour Power Bakery

A warm, fully responsive **one-page landing site** for *Flour Power Bakery* — an artisan bakery specialising in sourdough breads, butter pastries, celebration cakes, and handcrafted cookies.

---

## 📄 Pages & Sections

| Section | Description |
|---|---|
| **Nav** | Fixed, blur-backdrop navigation with smooth-scroll links |
| **Hero** | Full-viewport hero with background image, tagline, and dual CTAs |
| **Product Highlights** | 4-card grid showcasing signature baked goods with emoji art, tags & prices |
| **Why Us** | 4-pillar trust section (local ingredients, freshness, custom orders, community) |
| **Contact** | Side-by-side store info + functional enquiry form with submit feedback |
| **Footer** | Simple branded footer |

---

## 🛠️ Tech Stack

- **Pure HTML5 + CSS3** — zero frameworks, zero dependencies
- **Inline CSS** with CSS custom properties (design tokens)
- **Vanilla JS** — IntersectionObserver for active-nav highlighting; form submit feedback
- Fully **responsive** via CSS Grid + `clamp()` fluid typography

---

## 🚀 Running Locally

```bash
# Clone the repo
git clone https://github.com/Skarchenllc/flour-power-bakery.git
cd flour-power-bakery

# Open in browser (no build step needed)
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

---

## 🌐 Deployment

The site is a single static `index.html` file and can be deployed to:

- **GitHub Pages** — enable in repo Settings → Pages → `main` / `root`
- **Netlify / Vercel** — drag-and-drop the folder, done
- **Any static host** — no server-side dependencies
